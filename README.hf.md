# PDFMathTranslate

PDF 学术论文翻译工具，保留公式、图表、目录和注释。

## ✨ 功能特点

- 📊 保留文档格式（公式、图表、目录、注释）
- 🌐 支持多种语言翻译
- 🤖 支持多种翻译服务（OpenAI、DeepL、Google、SiliconFlow 等）
- 📄 生成单语/双语 PDF 对照版本

## 🚀 使用方法

1. 上传 PDF 文件
2. 选择翻译服务和目标语言
3. 点击翻译按钮
4. 下载翻译后的 PDF

## 📝 注意事项

- 免费版硬件资源有限，大文件可能翻译较慢
- 部分翻译服务需要配置 API 密钥
